# Сотрудники

Фулстак приложение с регистрацией и авторизацией.

React + Ts, Redux, RTK querry, Router, Atn Design.

---

Запуск Prisma DB

```bash
npx prisma studio
```

Запуск фронта

```bash
npm run dev
```

Запуск бэкэнда

```bash
npm run server
```

Запуск фронт + бэкэнд

```bash
npm run app
```
